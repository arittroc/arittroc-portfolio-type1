# Portfolio Project Overview

## Project: Arittro Chowdhury — Developer Portfolio

### Purpose
A high-end, professional developer portfolio showcasing technical expertise, projects, and skills for potential employers and collaborators. Built with modern web technologies emphasizing both aesthetics and performance.

---

## Project Goals

1. **Professional Branding** — Establish a strong personal brand identity with distinctive visual language
2. **Project Showcase** — Display 4 key projects with detailed descriptions and technical stack
3. **Skills Display** — Present technical capabilities organized by category
4. **Contact Pathways** — Provide multiple ways for potential collaborators/employers to connect

---

## Target Audience

- **Recruiters & Hiring Managers** — Evaluating technical skills and project experience
- **Fellow Developers** — Potential collaborators or networking contacts
- **Tech Companies** — Looking for MLOps/Cloud engineers
- **Design-Conscious Clients** — Appreciating premium web aesthetics

---

## Key Features

### Visual Design
- **Dark Theme** with amber (#F59E0B) accents
- **Instrument Sans** typography for readability
- **3D tilt effects** on project cards
- **Custom cursor** with smooth trail animation
- **Parallax scrolling** with Lenis smooth scroll

### Sections
1. **Hero** — Name, role, availability status, CTAs
2. **About** — Background, education, stats
3. **Projects** — 4 showcase projects with horizontal scroll (desktop)
4. **Skills** — Categorized tech stack with icons
5. **Footer** — Contact, social links, CTA

### Interactions
- Mouse-tracking 3D transforms on cards
- Smooth scroll navigation
- Hover animations and glow effects
- Mobile-responsive accordion layouts

---

## Technical Highlights

- **Next.js 15** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Custom cursor** with requestAnimationFrame
- **GSAP ScrollTrigger** for horizontal scroll
- **Lenis** for smooth scroll
